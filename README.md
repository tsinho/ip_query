# ip_query
Python电脑IP归属地本地查询小工具

### 需搭配数据库使用
- Pickle数据库：`https://www.123865.com/s/jUjUVv-vE7WA`
- Csv数据库：`https://www.123865.com/s/jUjUVv-dE7WA`

### 使用教程：
- 下载数据库命名`data.pickle`/`data.csv`放在同目录下即可（程序择优加载，优先加载pickle数据库）
- 数据来源ip2location`https://lite.ip2location.com/`

### 数据更新方法：
- 下载官方IP2Location™ LITE IP-COUNTRY-REGION-CITY-LATITUDE-LONGITUDE-ZIPCODE-TIMEZONE Database的`csv`格式数据库到主程序同目录下命名data.csv即可
- pickle数据库加载速度较快一些，也可以转换后命名data.pickle放在主程序同目录下
